# PokeComp Redesign (V2 — Apr 27 2026)

## Core Philosophy
Pro-tool esports-broadcast feel. The system prioritizes **scan speed over spectacle**, targeting 200ms data retrieval for users during live matches. The aesthetic is warmer and more confident than the original V1 "Battle Station" theme, using a magenta/gold/purple palette.

## Core Palette
- **Background:** `#0a0510` (Near-black)
- **Primary / Live / Danger:** `#FF2D7A` (Magenta)
- **CTA / Accent:** `#FFD23F` (Gold)
- **AI / Contextual:** `#7E22CE` (Purple)
- **Success:** `#22C55E` (Green)
- **Legacy Warning:** `#F59E0B` (Amber, used as tertiary color)

## Typography
- **Primary (Body + Display):** Inter.
- **Telemetry / IDs / Stats:** JetBrains Mono (used for ALL-CAPS labels, percentages, and numerical data).
- **Tracking:** Headlines use tight tracking (`-0.035em` or `-0.045em`).

## UI Components & Patterns
- **Mono Labels:** Section headers follow the `◆ LABEL · COUNT` pattern with `0.22em` letter-spacing.
- **Borders:** **Mandatory** 1px solid `outline-variant` on all cards, panels, and inputs.
- **Radii:** 
  - Cards: `0.875rem`
  - Buttons/Inputs: `0.5rem`
  - Badges/Dots: `9999px` (Full round)
- **Materiality:** Opaque panels with subtle radial-gradient halos. **No glassmorphism, no 3D perspective.**
- **Cards:**
  - `.card`: Opaque background `rgba(15,9,22,.6)` with outline border.
  - `.card-interactive`: Hover state lifts to a gold-tinted border.
- **Buttons:**
  - `.btn-primary`: Solid gold background with dark text.
  - `.btn-gradient`: Linear gradient (magenta → gold) for high-impact CTAs.
  - `.btn-ghost`: Transparent with 1px border.

## Brand Assets
- **Logo:** 30x30 conic-gradient tile (magenta → purple → gold) wrapping a near-black square with a gold "P".
- **Sprites:** 
  - Pokemon: Official artwork for detail pages; front sprites for lists.
  - Items: `/sprites/items/<slug>.png`.
- **Disclaimer:** All footers must include: *"A solo fan project. Not affiliated with The Pokemon Company, Nintendo, or Game Freak."*

## Implementation Reference
Tokens are managed via **Tailwind CSS v4** CSS-first `@theme` configuration in `web/src/app/globals.css`. Do not use `tailwind.config.ts`.
