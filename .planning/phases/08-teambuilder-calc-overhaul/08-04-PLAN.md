# Plan 08-04: Admin Triggers & Personal History RAG

## Goal
Empower admins to update meta data on demand and improve AI analysis by injecting user-specific performance metrics against meta threats.

## Proposed Changes

### 1. Admin UI (Manual Triggers)
- **Action:** Add a "Live Data" section to the Admin Dashboard with buttons: "Scrape Pikalytics", "Scrape Limitless".
- **Backend:** Implement `@router.post("/ingest/{source}")` endpoints that run ingest logic as `fastapi.BackgroundTasks`.
- **File:** `api/app/routers/admin.py`, `web/src/app/admin/page.tsx`

### 2. Personalized AI (RAG Upgrade)
- **Action:** Update the `analyze_draft` function to fetch win/loss stats from the `matchup_log` for the specific species in the opponent's team.
- **Logic:** Calculate `wins / total` for each species and inject into the Claude prompt as "Historical Performance" metadata.
- **File:** `api/app/routers/draft.py`
